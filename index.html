มาทำเว็บไทยให้ กันเถอะ

เราสนับสนุนให้เว็บไทยเปลี่ยนเป็น HTTPS

ความตั้งใจของ Secure The Web Thailand

แวดวงการพัฒนาเว็บไซต์ในต่างประเทศกำลังตื่นตัวและให้ความสำคัญกับความปลอดภัยของการรักษาข้อมูล จึงได้มีการผลักดันให้มีการติดตั้ง SSLในทุกๆเว็บไซต์ แรงผลักดันเหล่านี้ถูกขับเคลื่อนโดยกลุ่มผู้พัฒนา Browser เจ้าหลักๆในโลก (เช่น Google Chrome)ได้เริ่มทำการปิดกั้นการเข้าถึงคุณสมบัติใหม่ๆ จากเว็บไซต์ที่ไม่ได้มีการติดตั้ง SSL และเพื่อเป็นการสนับสนุนให้เว็บไซต์ในประเทศไทยใส่ใจในความปลอดภัยในข้อมูลของผู้ใช้มากขึ้น พวกเรา "Secure The Web Thailand" ขอเป็นตัวแทนในการผลักดันและขับเคลื่อนให้การเปลี่ยนผ่านไปสู่ยุค HTTPS เกิดขึ้นในประเทศไทยอย่างจริงจัง จนสามารถสร้างความมั่นใจให้กับผู้ใช้งานได้ว่าข้อมูลต่างๆ จะไม่รั่วไหลไปอยู่ในมือของผู้ไม่ประสง

ทำไมต้อง HTTPS?

มั่นใจว่าข้อมูลของผู้ใช้ปลอดภัย

การติดตั้ง SSL จะทำให้เจ้าของเว็บไซต์มั่นใจได้ว่าข้อมูลของผู้ใช้จะไม่ถูกดักจับจากผู้อื่นได้ระหว่างทาง

Browser จะไม่เตือนว่าเว็บไซต์ของคุณไม่ปลอดภัย

ใน Browser รุ่นใหม่ๆ ถ้าผู้ใช้เข้าเว็บไซต์ที่ไม่ได้ติดตั้ง SSL จะมีคำเตือนมาบอก User ทันที่ว่าเว็บไซต์นี้ไม่ปลอดภัย

มีผลต่อ SEO

Search Engine หลายๆเจ้ากำลังจะเริ่มให้ความสำคัญกับ HTTPS และจะนำมานับรวมในคะแนนผลการค้นหาเร็วๆนี้

ได้ใช้ Feature ใหม่ๆ

Feature ใหม่ๆใน มาตรฐาน HTML จะไม่สามารถใช้งานบนเว็บไซต์ ที่ไม่ได้ติดตั้ง SSL รวมถึง Feature เก่าๆจะเริ่มไม่สามารถใช้งานบน HTTP ได้เช่นเดียวกัน
